## Opis domene
Programska potpora za rad ljekarne. Ovu aplikaciju bi koristile zaposlene osobe unutar ljekarne. Postojale bi dvije uloge - admin i zaposlenik. Osnovna domena rada aplikacije bi bila- praćenje količine i vrste lijekova, naručivanje lijekova, izdavanje računa, dodavanje novih lijekova u bazu, upravljanje korisničkim računima zaposlenika, pregled mjesečnih izvadaka, zarede...
